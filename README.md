# SQL Server Health Check

A PowerShell-based health check tool for Microsoft SQL Server.

## Overview

SQL Server Health Check is an open-source project that helps administrators quickly collect configuration and operational information from SQL Server environments.

The goal is to simplify routine health checks and support troubleshooting activities.

## Planned Features

### SQL Server Information

- SQL Server version
- Edition
- Instance name

### Memory Configuration

- Max Server Memory
- Min Server Memory

### Database Information

- Database list
- Database size
- Recovery model

### Agent Jobs

- SQL Agent Job list
- Job status

### Index Information

- Missing indexes
- Fragmented indexes

## Technologies

- PowerShell 7
- SQL Server
- T-SQL
